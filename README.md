# Project-A.R.K---Solomon-AI

This project combines a React frontend with backend services to provide an AI-powered chat experience.

## Structure

- `frontend/`: The React frontend application.
- `backend-python/`: (Optional) Python backend (e.g., using Flask/FastAPI) for specific logic or AI integration.
- `backend-laravel/`: PHP backend using Laravel, providing the main API endpoints.

## Running the Project

- Frontend (React): Navigate to `my-ai-chatbot/` and run `npm start`.
- Backend (Laravel): Navigate to `backend-laravel/` and run `php artisan serve`.
- Backend (Python): Navigate to `backend-python/` and run the appropriate startup command (e.g., `python app.py`).
